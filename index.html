Overview

What is Graphqlator?

Graphqlator generates a GraphQL API implemented in Go.

How does it work?

Graphqlator uses your existing database table schema(s) to generate the server.

Getting Started

Installation

Install using go get

go get github.com/ahmedalhulaibi/graphqlator

Download Pre-Built Binaries from GitHub

Prerequisites

Mandatory prerequisites:

  • Go
  • GORM
  • GraphQL Go

One of the following databases:

  • MySQL
  • MariaDB
  • PostgreSQL
